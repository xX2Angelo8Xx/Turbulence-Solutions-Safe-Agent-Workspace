# Mathematik Demo (Streamlit)

Kurze interaktive Mathematik-Demos in Python mit Streamlit.

Setup:

1. Virtuelle Umgebung erstellen (optional):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Abhängigkeiten installieren:

```powershell
pip install -r requirements.txt
```

3. App starten:

```powershell
streamlit run app.py
```

Inhalt:
- `app.py`: Streamlit-App mit zwei kleinen Demos (Quadratische Funktion, Sinuswellen).# Project
