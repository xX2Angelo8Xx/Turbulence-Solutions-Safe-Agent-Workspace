## AgentDocs

The `AgentDocs/` subfolder contains shared knowledge for both humans and agents:

- `AGENT-RULES.md` — Complete agent permissions, tool rules, and known workarounds
- `progress.md` — Current project state (read this at session start)
- `plan.md` — Active task breakdown
- `architecture.md` — System design and tech stack
- `decisions.md` — Key decisions and rationale
- `research-log.md` — Findings and references
- `open-questions.md` — Unresolved trade-offs

## Quick Tips

- Agents read `progress.md` at the start of each session
- All agent decisions are logged in `decisions.md`
- Keep `progress.md` up to date as work is completed
